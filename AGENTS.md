# AGENTS.md — DjavaLauncher-2.0 (GTA SA Reversed Android)

## What this is

Single-module Android app (`:app`, namespace `com.nathan.djavarp`) that is a reverse-engineered **SAMP Mobile** launcher/client for GTA: San Andreas.  
MinSdk 26, targetSdk 36, compileSdk 34.  
Root Gradle project name: `gtareversed`.

## Build commands

```bash
./gradlew assembleDebug        # Debug APK (uses release keystore, minify=false)
./gradlew assembleRelease      # Release APK
```

- AGP 8.2.1, Gradle wrapper 8.9, NDK 26.2.11394342.
- Both debug & release use `keystore_playmarket2.jks` (`hayk2010`).
- Post-build: automatically copies APKs to `C:\laragon\www`.
- Lint: `abortOnError = false` — won't fail on lint warnings.

## ABI splits

`armeabi-v7a` + `arm64-v8a` only (no x86, no universal APK).  
Pre-built `.so` files live under `app/src/main/jniLibs/`.

## Native (C++) layer

- CMake minimum 3.12, C++20.
- Project `reversed` at `app/src/main/cpp/CMakeLists.txt`.
- Submodules: `samp/` (SA-MP client implementation) and `opus/` (audio codec).
- **Shadowhook** for function hooking (required via CMake `find_package`).
- Also links: BASS audio (`libbass.so`, `libbass_ssl.so`), GlossHook, EGL, GLESv3.
- Debug builds: `-fvisibility=default`; Release: `-fvisibility=hidden`.

## Java/Kotlin entry points

| Class | Role |
|---|---|
| `SplashScreenActivity` | Launcher activity (portrait) |
| `MainActivity` | Bottom-nav host (Home/Server/Download/Settings) |
| `SAMP` (extends `GTASA`) | Game activity (landscape), native bridge |
| `DjavaApplication` | App init (PRDownloader) |

## Packages

| Package | Contents |

## ProGuard / Paranoid

- Obfuscation via Paranoid (`@Obfuscate` on `SAMP.java`).
- ProGuard keeps all classes in `com.nvidia.devtech.*`, `com.wardrumstudios.utils.*`, `com.nathan.djavarp.game.*`.

## CI

`.github/workflows/c-cpp.yml` is **autogenerated and non-functional** — it runs `./configure && make` but this repo has no such scripts. Ignore it.

## Game data downloader

`downloader.py` fetches assets from `https://samp-mobile.shop/files.json` into `gamedata/`.  
Resumes by file-size matching. Call it separately; not part of the Gradle build.

## Dependencies

Version catalog at `gradle/libs.versions.toml`. Notable:
- Firebase BOM 33.6.0 (Analytics, Crashlytics NDK, Messaging)
- Shadowhook 1.0.9 (ByteDance)
- ini4j for `settings.ini` (SAMP server config)
- Local jar: `app/libs/com.bda.controller.jar` (NVIDIA controller)
- Volley, PRDownloader, Glide, Retrofit2+Gson

- `android:gwpAsanMode="always"` in manifest.
